This is my portfolio website, a dedicated place to showcase all of my skills, experience, and projects in detail.

Whether you're a potential employer, collaborator, or just curious, this portfolio provides a comprehensive look at my work and aspirations in the tech industry.

## How to run on your machine

1. Run the command `npm start` in command line / terminal.
2. Open `http://localhost:3000` in your preferred browser.
